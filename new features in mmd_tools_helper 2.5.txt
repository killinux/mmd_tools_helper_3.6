MMD Tools Helper 2.5 新功能和更新

版本 2.5 主要更新：

1. Blender 3.6 兼容性
   - 完全兼容 Blender 3.6.0 及更高版本
   - 不再支持 Blender 2.79b 或更早版本
   - 所有功能已更新以使用最新的 Blender Python API

2. UI 界面更新
   - 所有面板现在位于 3D 视图的侧边栏（N 面板）中
   - 面板位置从旧的工具架迁移到现代的 UI 区域
   - 保持相同的面板类别 "mmd_tools_helper"

3. API 现代化
   - 更新了插件注册系统，使用个别类注册方法
   - 修复了所有废弃的 Blender Python API 调用
   - 使用 importlib 替代废弃的 imp 模块
   - 更新了场景对象操作 API
   - 更新了用户偏好设置 API
   - 修复了灯光系统 API 变更

4. 功能保持
   - 所有原有功能完全保持不变
   - MMD View、MMD Lamp、骨骼重命名等工具正常工作
   - 材质节点创建和显示面板组功能正常
   - 杂项工具集合功能完整

5. 技术改进
   - 代码结构更加符合现代 Blender 插件标准
   - 改进了错误处理和兼容性
   - 优化了插件加载和卸载过程

安装说明：
1. 确保使用 Blender 3.6.0 或更高版本
2. 在 Blender 中安装插件
3. 在 3D 视图中按 N 键打开侧边栏
4. 查找 "mmd_tools_helper" 标签页

注意事项：
- 如果从旧版本升级，请先卸载旧版本插件
- 面板位置已从工具架移动到侧边栏
- 所有快捷键和操作方式保持不变
